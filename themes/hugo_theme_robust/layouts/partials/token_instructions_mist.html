Mist

If this is your first token purchase, we recommend that you complete it using MyEtherWallet. If you have experience using Mist, please follow the instructions shown below:

  1. Open your Mist app.
  2. Go to the “Send” tab in the options.
  3. Paste the adToken contract address found above in the “To:” section.
  4. The fee should be set to the faster end of the bar. Click “Send”.
  5. A summary of the transaction will appear. Please note, the transaction has not yet been generated.
  6. Enter your password again and click “Send Transaction”.
