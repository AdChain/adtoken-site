Parity

If this is your first token purchase, we recommend that you complete it using MyEtherWallet. If you have experience using Parity, please follow the instructions shown below:

  1. Make sure you have the latest version of Parity installed.

  2. Start Parity application and navigate to http://127.0.0.1:8180 in your browser.

  3. Enable the Contracts tab by checking the checkbox where it say "Contracts".

  4. Go to "CONTRACTS" tab and click on "+ WATCH" menu item.

  5. Select "Custom Contract" option and click the "Next" button.

  6. In the "ENTER CONTRACT DETAILS" form, enter the following:

    • In "network address" field, enter adToken Contract Sale Address found above.

    • In "contract name" field, enter "AdToken Sale".

    • In "contract abi" field, enter adToken Contract Sale ABI JSON found above.

    Then click on "+ ADD CONTRACT" button.

  7. You will now see the newly added contract under "CONTRACTS". Click on "AdToken Sale" contract.

  8. Click on "EXECUTE" button.

  9. In the "FUNCTION DETAILS" form, set "function to execute" dropdown to "purchaseTokens()"

  10. In the "transaction value (in ETH)" field, enter the desired of ETH you'd like to use to purchase adToken.

  11. Click on "POST TRANSACTION" button.

  12. Enter Account Password to unlock account and click on "CONFIRM REQUEST" button.

  13. You can check the status of the purchase in etherscan website.
