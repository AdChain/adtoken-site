CONTRACT FUNCTION TO EXECUTE
purchaseTokens()

The purchaseTokens() smart contract function needs to be executed in order to purchase tokens. You must send ETH when calling purchaseToken() function to buy adToken. Further instructions are detailed below.

IMPORTANT NOTES

  • Do NOT send ETH directly to the adToken sale contract address. Please follow instructions below on how to execute purchaseTokens() smart contract function in order to purchase adToken.
  • Do NOT send funds from exchanges such as Poloniex, Bitfinex, Bittrex, Coinbase, GDAX, etc.
  • Do not send your funds before the fundraiser has begun.
  • We recommend using the wallet applications mentioned below.
  • Only ETH is accepted. Do NOT try sending anything other than ETH.
  • ALWAYS double check addresses you are sending ether to.
